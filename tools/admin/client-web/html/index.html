Intranet Administration

Permission Assignment

Each tool on the intranet has a set of privileges that can be assigned.

  • Privileges determine what a user or group can do with a tool.
  • Each tool defines its own set of privileges that it will use.
  • All tools have an access-tool privilege that determines who can use the tool in the first place.
  • Privileges should almost always be assigned at the group level.
