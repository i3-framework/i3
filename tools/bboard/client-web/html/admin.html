About Permissions

Each bulletin board topic has a set of permissions. A permission provides a set of privileges to an individual user, a group of users, or all users of a topic. The following privileges can be assigned:

  • View
    Users with this privilege can see the topic in lists and read the posts and comments that have been made in the topic.
  • Post
    Users with this privilege can post new messages in the topic.
  • Comment
    Users with this privilege can comment on existing messages.
  • Moderate
    Users with this privilege can modify or delete the messages and comments that have been posted by others. This privilege can only be granted to specific users or groups.

A user is considered to have a privilege if the user is a member of any group that has the privilege. For example, if a topic is viewable by everyone, and the topic also has a permission entry for an individual user, it does not matter if the user specifically has the view privilege or not; the user will automatically gain the privilege that all users have.
